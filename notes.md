# Site du club d'Intelligence Artificielle de l'UdeM


To create a new post: `hugo new posts/<newpost>.md`

To download modules
```
hugo mod init randomName
hugo mod vendor
hugo serve -D
```

To build website:
```
hugo -d public/
```
